Error Handling CHAPTER 6

Every function returns a Result<T, E>. Read more in the Maybe section, then try the example below.

# a function that can fail
fn halve(i32 n) i32:
    if (n % 2 != 0):
        return Result.Err(MathError.Odd())
    return Result.Ok(n / 2)
Note · don't use ?? in main()
Inside main() handle the Result explicitly with a match or .realise(default).
